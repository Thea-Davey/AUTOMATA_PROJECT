OPERATING MANUAL

1. INSERTING CARTRIDGES

Click on REGEX 1 or REGEX 2 to slot the cartridge into the console. Click again to eject. The system must have a cartridge loaded to process strings.

2. SELECTING A MODEL

Once a cartridge is loaded, select your visualization mode:

  • DFA: Deterministic Finite Automaton
  • CFG: Context-Free Grammar
  • PDA: Pushdown Automaton

3. STRING SIMULATION

Type a string (no spaces) into the input row and press VALIDATE. The TV screen will track the string's journey. Use + ADD MORE to test multiple strings.

4. THE REGEX DATABASES

REGEX 1: (b+aa+ab) (a+b)* (bb+aba+ab)* (aaa+bbb) (a+b) (a+b+ab)*
Alphabet: {a, b}


REGEX 2: (1+0)* (11+00+101+010) (1+0+11+00+101)* (11+00) (11+00+101)* (1+0) (1+0+11)*
Alphabet: {0, 1}
